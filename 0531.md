#### 5. OmniVLA-Edge 아키텍처 분석

**결론:**
- `train.py`의 `IL_gps_map_mask3_lan2` = inference의 `OmniVLA_edge` — 동일한 아키텍처
- `train_utils.py`의 학습 함수는 3개 DataLoader 동시 사용, rides_11에 직접 연결 불가
- **별도 `finetune_omnivla_edge.py` 작성이 올바른 방향**

**forward 시그니처:**
```python
model(
    obs_img,      # (B, 18, 96, 96)  — 6프레임 × 3ch
    goal_pose,    # (B, 4)           — [x, y, cos, sin]  (rides_11: zeros)
    map_images,   # (B, 9, 96, 96)   — OSM(3)+OSM(3)+obs(3)
    goal_img,     # (B, 3, 96, 96)   — obs_cur 반복     (rides_11: obs_cur)
    goal_mask,    # (B,)             — modality_id=0 "satellite only"
    feat_text,    # (B, 512)         — CLIP 피처        (rides_11: zeros)
    current_img,  # (B, 3, 224, 224) — FiLM용 고해상도
)
→ action_pred (B, 8, 4), dist_pred (B, 1)
```

### Option B: odom map (3ch) 으로 goal_encoder 교체

**목표:** OSM 맵(9ch) 대신 odom map 한 장(3ch)만 goal_encoder 입력으로 사용.
모델 구조를 변경하므로 weight 변환 → 모델 파일 수정 → dataset 수정 → finetune 스크립트 수정 순으로 진행.

**구조 검증 완료:**
```
goal_encoder._conv_stem.weight  체크포인트 shape: (32, 9, 3, 3)  ✓
bias: None (변환 불필요)
include_top: True
```

**forward 내 map_encoding 흐름 (변경 없음):**
```
extract_features(map_images)  (B, 1280, 3, 3)
.unsqueeze(1)                 (B, 1, 1280, 3, 3)
obs_encoder._avg_pooling()    (B, 1, 1280, 1, 1)
flatten(start_dim=1)          (B, 1280)
compress_obs_enc_map()        (B, 1024)
.unsqueeze(1)                 (B, 1, 1024)  ← 토큰으로 concat
```
→ map_images 채널 수만 바뀌고 이후 흐름 동일.

#### Step 1 완료 — `convert_goal_encoder_9ch_to_3ch.py`

경로: `Learning-to-Drive-Anywhere-with-MBRA/convert_goal_encoder_9ch_to_3ch.py`

**변환 전략:**
- `goal_encoder._conv_stem.weight`: `(32, 9, 3, 3)` → `(32, 3, 3, 3)`
- 채널 구성 `OSM(0-2) | OSM(3-5) | obs(6-8)` → 3그룹 평균
  ```python
  w3 = (w9[:, 0:3] + w9[:, 3:6] + w9[:, 6:9]) / 3.0
  ```
- 나머지 모든 key: shape 변화 없음, 1:1 복사
- 검증: 변환 후 3ch 모델에 `load_state_dict(strict=False)` + 더미 forward 확인

**실행 명령 (host, mbra 환경):**
```bash
conda run -n mbra python convert_goal_encoder_9ch_to_3ch.py \
  --input  /media/ms/WD_BLACK_4TB/OmniVLA/omnivla-edge/omnivla-edge.pth \
  --output /media/ms/WD_BLACK_4TB/OmniVLA/omnivla-edge/omnivla-edge-odom3ch.pth
```

**출력:** `/media/ms/WD_BLACK_4TB/OmniVLA/omnivla-edge/omnivla-edge-odom3ch.pth`
